Task Driven vs Data Driven

Depending on what your intended outcome of machine learning is, there are two main routes you can go down. The journey usually gets decided upon the type of data you have. If you have data that is labelled, you can take the task-driven course, but if your data is unlabelled, then you can go down the data-driven route.
